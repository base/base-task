# base-task [![NPM version](https://img.shields.io/npm/v/base-task.svg)](https://www.npmjs.com/package/base-task) [![Build Status](https://img.shields.io/travis/node-base/base-task.svg)](https://travis-ci.org/node-base/base-task)

> base plugin that provides a very thin wrapper around [https://github.com/doowb/composer](https://github.com/doowb/composer) for adding task methods to your application.

- [Install](#install)
- [Usage](#usage)
- [License](#license)

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm i base-task --save
```

## Usage

```js
var Base = require('base-methods');
var tasks = require('base-task');
Base.use(tasks());

var base = new Base();
base.task('foo', function(cb) {
  console.log('this is foo!');
  cb();
});
base.task('bar', function(cb) {
  console.log('this is bar!');
  cb();
});

base.build(['foo', 'bar'], function(err) {
  // this is foo!
  // this is bar!
  console.log('done!');
});
```

**v0.3.0**

* Bumped [composer](https://github.com/jonschlinkert/composer) to v0.11.0, so the `.watch` method is no longer included by default. To add `.watch`, use the [base-watch](https://github.com/node-base/base-watch) plugin.
